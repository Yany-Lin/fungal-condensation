Freehand Boundary Calibration

1 · Load image
2 · Set scale
3 · Trace boundary
4 · Export
Use circle-dragger tool for elliptical sources · use this tool for irregular/elongated boundaries

1 · Image

Load the reference frame (e.g. 15m00s.jpg). The image is never uploaded — everything runs locally.

2 · Scale calibration

Click two points on the scale bar, then enter the known distance.

3 · Trace boundary

Freehand: hold mouse and drag to trace. Release to close automatically.

4 · Export

Outputs the same calibration.json schema used by the Python pipeline (polygon_x / polygon_y + fitted ellipse).
